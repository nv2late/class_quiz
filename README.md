## 기록하는 방식

- 시안이나 제시된 문제에 따라서 스스로 기능을 구현해보고 있습니다.
- 완전 몰라서 검색해서 완성한 내용이나 구체적인 부분은 노션에 따로 기록중이기에 코드에 주석으로 적어두지 않습니다.
- 더 좋은 방법을 찾지 못했거나 추후에 찾고자 하지만 급하지 않은 경우에는 주석처리를 해놓습니다.
